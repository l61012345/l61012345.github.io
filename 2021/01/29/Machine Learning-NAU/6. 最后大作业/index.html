Beginner Level

For this level we use the Chinese Traffic Sign Database (Traffic Sign Recogntion Database (ia.ac.cn)). This is available as a zip file in your project folder under the name â€œDataset_1.zipâ€.   • Use the sklearn SVM package to train a classifier using x_train and y_train.
